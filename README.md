# Crowdfunding Analytics Project

## Project Overview
This Crowdfunding Analytics Project was developed to analyze crowdfunding campaign performance and uncover meaningful insights using data visualization and business intelligence tools. The project focuses on campaign success analysis, funding trends, backer engagement, and category-wise performance using interactive dashboards and KPI tracking.

The project was developed collaboratively as a team project using Excel, SQL, Power BI, and Tableau.

# Project Objectives
- Analyze crowdfunding campaign performance
- Track successful and failed projects
- Identify top-performing categories and locations
- Measure funding success rates
- Monitor backer engagement and campaign trends
- Build interactive dashboards for business insights

---

# Data Preparation & Transformation
### 1. Epoch Time Conversion
- Converted Epoch Date fields into Natural Date format for analysis.

### 2. Calendar Table Creation
Created a dynamic Calendar Table using the Created Date column with:
- Year
- Month Number
- Month Full Name
- Quarter
- Year-Month
- Weekday Number
- Weekday Name
- Financial Month
- Financial Quarter

### 3. Data Modeling
- Built relationships between multiple Excel files.
- Created a structured data model for dashboard analysis.

### 4. Currency Conversion
- Converted Goal Amount into USD using static USD conversion rates.

# Key Performance Indicators (KPIs)

## Project Overview KPIs
- Total Number of Projects based on Outcome
- Total Number of Projects based on Location
- Total Number of Projects based on Category
- Total Number of Projects created by Year, Quarter, and Month

## Successful Projects KPIs
- Total Amount Raised
- Total Number of Backers
- Average Number of Days for Successful Projects

## Top Successful Projects
- Top Projects based on Number of Backers
- Top Projects based on Amount Raised

## Success Percentage Analysis
- Overall Percentage of Successful Projects
- Success Percentage by Category
- Success Percentage by Year and Month
- Success Percentage by Goal Range
